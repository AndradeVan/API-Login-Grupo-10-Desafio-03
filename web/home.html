description Sobre o Desafio

Você continuará no mesmo grupo do desafio 3 e juntos terão que construir uma Aplicação Web que consome a API que o grupo construiu no desafio 3. Além disso, terão que construir (sem uso de IA Generativa) os seus testes com Cypress para avaliar os 4 cenários propostos no desafio 3.

Seu projeto deve considerar a criação de um site, usando as tecnologias:

  • style CSS
